# Handwritten Digit Recognizer
This project is part of my **Artificial Intelligence Internship**. It uses a **Convolutional Neural Network (CNN)** to identify handwritten digits (0-9).

### 🚀 Highlights
* **Dataset**: MNIST (70,000 images)
* **Model**: Deep Learning (CNN) built with TensorFlow/Keras
* **Accuracy**: Achieves over 98% accuracy
* **Tools**: Google Colab, Python, Matplotlib
